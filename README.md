# Analyzer

Простой анализатор DHCP-трафика на Python с использованием pyshark.
Парсит pcap/pcapng файлы, собирает статистику и строит базовые графики.

## Что делает

- Извлекает DHCP-пакеты из файла захвата
- Определяет типы сообщений (Discover, Offer, Request, ACK и др.)
- Выделяет hostname клиентов и выданные IP-адреса
- Строит два графика: распределение по типам и временную последовательность
- Формирует текстовый отчёт в консоль и в файл

## Зависимости

- Заработало только на Python 3.7.3 c обновленным pip до версии 18
- Wireshark (tshark)
- Python-библиотеки из `requirements.txt`
